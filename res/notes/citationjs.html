How to install citation.js

The guides make it sound like you need to install through npm (which I can't), but that turns out not to be the case.

I found the citation.js-file (probably the one at https://cdn.jsdelivr.net/gh/larsgw/citation.js@archive/citation.js/citation-0.4.0-9.js), saved it. I could not make the bundles with plugins work (something about the Cite constructor not being a constructor...).  

Then I followed this template: https://runkit.com/larsgw/591b5651bd9b40001113931c but to use multiple Wikidata IDs (Q), you have to put them in square brackets. E.g. let example = new Cite(['Q121071205','Q106128941']).

This means that the HTML page with the citation contains the following in the <head>-tag:

<script>
        const Cite = require('citation-js')

        let example = new Cite(['Q121071205','Q106128941'])

        let output = example.format('bibliography', {
        format: 'html',
        template: 'apa',
        lang: 'en-US'
        })
    </script>

and then on the page, where i want citation to appear, I have:

 <script>

document.write(output);
</script>
